# bx-js-mongo
Nodejs Mongo App

## Installations

- run `npm i`

start the mongo server or mysql server

- run `node index.js`



## apiKeys Endpoints

### Base URL : `<host>/apiKeys`

- *GET* `/`
    Gets all apiKeys

- *POST* `/`
    Create a new apiKey on the database

- *GET* `/:apiKey`
    Fetches the apiKey with `id` `:apiKey`

- *PUT* `/:apiKey`
    Modifies the apiKey with `id` `:apiKey`

- *DELETE* `/:apiKey`
    Deletes the apiKey with `id` `:apiKey`
